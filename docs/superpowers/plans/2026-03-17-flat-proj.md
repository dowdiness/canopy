# FlatProj Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the nested Let spine in the projection layer with a flat array, eliminating cascading spine rebuilds during reconciliation.

**Architecture:** FlatProj stores let-defs as a flat `Array[(String, ProjNode, Int, NodeId)]` + optional final expression. The memo pipeline derives FlatProj instead of nested ProjNode. Backwards-compat conversions (FlatProj <-> ProjNode) keep tree_editor and tree_edit_bridge working without rewrite.

**Tech Stack:** MoonBit, moon test/check/info/fmt

**Spec:** `docs/plans/2026-03-15-projection-incremental-updates.md`

**Scope note:** This plan implements the FlatProj data structure, flat reconciliation (LCS on def names), and backwards-compat integration. Two spec items are deferred to follow-up tasks: (1) `physical_equal` CST comparison for O(changed_subtree) content edits — requires storing CST refs in FlatProj; (2) native flat-array tree edit operations — currently round-trips through ProjNode. The structural foundation is correct and the follow-ups are additive optimizations.

///|
/// Build a FlatProj from a SourceFile SyntaxNode.
/// Iterates LetDef children and collects them as flat array entries.
/// Each init subtree is built via syntax_to_proj_node (unchanged).
pub fn to_flat_proj(root : @seam.SyntaxNode, counter : Ref[Int]) -> FlatProj {
  let defs : Array[(String, ProjNode, Int, NodeId)] = []
  let mut final_proj : ProjNode? = None
  for child in root.children() {
    if @parser.LetDefView::cast(child) is Some(v) {
      let init = match v.init() {
        Some(n) => syntax_to_proj_node(n, counter)
        None =>
          error_node_for_syntax("missing let binding value", child, counter)
      }
      let id = NodeId(next_proj_node_id(counter))
      defs.push((v.name(), init, child.start(), id))
    } else if final_proj is None {
      final_proj = Some(syntax_to_proj_node(child, counter))
    }
  }
  { defs, final_expr: final_proj }
}
```

///|
/// Reconcile two FlatProjs, preserving node IDs where entries match.
/// Uses LCS on def names for alignment; reconciles init subtrees individually.
pub fn reconcile_flat_proj(
  old : FlatProj,
  new : FlatProj,
  counter : Ref[Int],
) -> FlatProj {
  let old_defs = old.defs
  let new_defs = new.defs
  let old_len = old_defs.length()
  let new_len = new_defs.length()

  // LCS DP on def names
  let dp : Array[Array[Int]] = []
  for i = 0; i <= old_len; i = i + 1 {
    let row : Array[Int] = []
    for j = 0; j <= new_len; j = j + 1 {
      row.push(0)
    }
    dp.push(row)
  }
  for i = 1; i <= old_len; i = i + 1 {
    for j = 1; j <= new_len; j = j + 1 {
      if old_defs[i - 1].0 == new_defs[j - 1].0 {
        dp[i][j] = dp[i - 1][j - 1] + 1
      } else {
        dp[i][j] = @cmp.maximum(dp[i - 1][j], dp[i][j - 1])
      }
    }
  }

  // Backtrace to find matched pairs
  let new_matched : Array[Int] = []
  for _ = 0; _ < new_len; _ = _ + 1 {
    new_matched.push(-1)
  }
  let mut i = old_len
  let mut j = new_len
  while i > 0 && j > 0 {
    if old_defs[i - 1].0 == new_defs[j - 1].0 {
      new_matched[j - 1] = i - 1
      i = i - 1
      j = j - 1
    } else if dp[i - 1][j] >= dp[i][j - 1] {
      i = i - 1
    } else {
      j = j - 1
    }
  }

  // Build reconciled defs
  let result_defs : Array[(String, ProjNode, Int, NodeId)] = []
  for j = 0; j < new_len; j = j + 1 {
    let (name, new_init, start, _new_id) = new_defs[j]
    if new_matched[j] >= 0 {
      let old_entry = old_defs[new_matched[j]]
      let reconciled_init = reconcile_ast(old_entry.1, new_init, counter)
      result_defs.push((name, reconciled_init, start, old_entry.3))
    } else {
      let fresh_init = assign_fresh_ids(new_init, counter)
      let fresh_id = NodeId(next_proj_node_id(counter))
      result_defs.push((name, fresh_init, start, fresh_id))
    }
  }

  // Reconcile final_expr
  let final_expr = match (old.final_expr, new.final_expr) {
    (Some(old_fe), Some(new_fe)) => Some(reconcile_ast(old_fe, new_fe, counter))
    (None, Some(new_fe)) => Some(assign_fresh_ids(new_fe, counter))
    _ => new.final_expr
  }

  { defs: result_defs, final_expr }
}
```

///|
/// Print a FlatProj to text. Equivalent to @ast.print_term on the nested spine.
pub fn print_flat_proj(fp : FlatProj) -> String {
  let parts : Array[String] = []
  for def in fp.defs {
    let (name, init, _, _) = def
    parts.push("let " + name + " = " + @ast.print_term(init.kind))
  }
  match fp.final_expr {
    Some(expr) => parts.push(@ast.print_term(expr.kind))
    None => parts.push("()")
  }
  parts.join("\n")
}
```

Note: The `None` → `"()"` matches `@ast.print_term(Unit)`. Verify the equivalence test passes — `@ast.print_term(Let(x, init, Let(y, init2, body)))` produces `let x = ...\nlet y = ...\nbody` which should match the flat version.

///|
/// Reconstruct a nested Let ProjNode spine from FlatProj.
/// Uses stored NodeIds for Let nodes. Counter only used for Unit fallback.
pub fn FlatProj::to_proj_node(self : FlatProj, counter : Ref[Int]) -> ProjNode {
  let mut result = match self.final_expr {
    Some(expr) => expr
    None => {
      // Use last def's end position so the synthetic Unit node doesn't
      // interfere with SourceMap position queries at offset 0.
      let end_pos = if self.defs.length() > 0 {
        self.defs[self.defs.length() - 1].1.end
      } else {
        0
      }
      ProjNode::new(Unit, end_pos, end_pos, next_proj_node_id(counter), [])
    }
  }
  for i = self.defs.length() - 1; i >= 0; i = i - 1 {
    let (name, init, def_start, id) = self.defs[i]
    result = ProjNode::new(
      Let(name, init.kind, result.kind),
      def_start,
      result.end,
      id.0,
      [init, result],
    )
  }
  result
}

///|
/// Extract a FlatProj from a nested ProjNode Let spine.
/// Walks the Let chain collecting entries until a non-Let node is found.
pub fn FlatProj::from_proj_node(root : ProjNode) -> FlatProj {
  let defs : Array[(String, ProjNode, Int, NodeId)] = []
  let mut current = root
  while true {
    match current.kind {
      Let(name, _, _) =>
        if current.children.length() >= 2 {
          defs.push((name, current.children[0], current.start, NodeId(current.node_id)))
          current = current.children[1]
        } else {
          break
        }
      _ => break
    }
  }
  let final_expr = match current.kind {
    Unit => None
    _ => Some(current)
  }
  { defs, final_expr }
}
```

```moonbit
fn build_projection_memos(
  rt : @incr.Runtime,
  source_text : @incr.Signal[String],
  syntax_tree : @incr.Signal[@seam.SyntaxNode?],
  prev_flat_proj_ref : Ref[@proj.FlatProj?],
  next_id_ref : Ref[Int],
) -> (
  @incr.Memo[@proj.FlatProj?],
  @incr.Memo[Map[@proj.NodeId, @proj.ProjNode]],
  @incr.Memo[@proj.SourceMap],
) {
  let proj_memo : @incr.Memo[@proj.FlatProj?] = @incr.Memo::new(
    rt,
    fn() -> @proj.FlatProj? {
      if source_text.get() == "" {
        prev_flat_proj_ref.val = None
        return None
      }
      let syntax_root = match syntax_tree.get() {
        Some(root) => root
        None => {
          prev_flat_proj_ref.val = None
          return None
        }
      }
      let counter = Ref::new(next_id_ref.val)
      let new_fp = @proj.to_flat_proj(syntax_root, counter)
      let reconciled = match prev_flat_proj_ref.val {
        None => new_fp
        Some(prev) => @proj.reconcile_flat_proj(prev, new_fp, counter)
      }
      next_id_ref.val = counter.val
      prev_flat_proj_ref.val = Some(reconciled)
      Some(reconciled)
    },
    label="proj_node",
  )

  let registry_memo : @incr.Memo[Map[@proj.NodeId, @proj.ProjNode]] = @incr.Memo::new(
    rt,
    fn() -> Map[@proj.NodeId, @proj.ProjNode] {
      let registry : Map[@proj.NodeId, @proj.ProjNode] = {}
      match proj_memo.get() {
        Some(fp) => {
          let counter = Ref::new(next_id_ref.val)
          @proj.register_flat_proj(fp, registry, counter)
        }
        None => ()
      }
      registry
    },
    label="proj_registry",
  )

  let source_map_memo : @incr.Memo[@proj.SourceMap] = @incr.Memo::new(
    rt,
    fn() -> @proj.SourceMap {
      match proj_memo.get() {
        Some(fp) => {
          let counter = Ref::new(next_id_ref.val)
          @proj.SourceMap::from_flat_proj(fp, counter)
        }
        None => @proj.SourceMap::new()
      }
    },
    label="source_map",
  )

  (proj_memo, registry_memo, source_map_memo)
}
```

```moonbit
pub fn SyncEditor::apply_tree_edit(
  self : SyncEditor,
  op : @proj.TreeEditOp,
  timestamp_ms : Int,
) -> Result[Unit, String] {
  let old_text = self.get_text()
  let old_cursor = self.get_cursor()

  // Get FlatProj, convert to ProjNode for tree edit
  let fp = match self.get_flat_proj() {
    Some(fp) => fp
    None => return Err("No FlatProj available")
  }
  let proj = fp.to_proj_node(self.next_node_id)

  let new_proj = match
    @proj.apply_edit_to_proj(
      proj,
      op,
      self.registry_memo.get(),
      self.next_node_id,
    ) {
    Ok(updated) => updated
    Err(msg) => return Err(msg)
  }

  // Convert result back to FlatProj
  let new_fp = @proj.FlatProj::from_proj_node(new_proj)
  let new_text = @proj.print_flat_proj(new_fp)

  if old_text == new_text {
    return Ok(())
  }

  // Seed FlatProj for next reconciliation
  self.seed_flat_proj(Some(new_fp))
  self.set_text_and_record(new_text, timestamp_ms)
  self.move_cursor(old_cursor)
  Ok(())
}
```
